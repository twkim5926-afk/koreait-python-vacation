#  셀리니움  vs requests
- requests: HTTP 통신결과를 파이썬으로 가공
- 페이지가 처음 로딩된 HTML만 가져옴
- 천천히 로딩되는 사이트는 크롤링 x

- Selenium: 내 pc를 조작하는 라이브러리
- 내 pc를 사용해서 인터넷창 키고 클릭 스크롤..
- 천천히 로딩되는 사이트도 크롤링 o
