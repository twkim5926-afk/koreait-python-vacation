## 라이브러리?
- 이미 다른 사람이 만들어 놓은 파이썬 패키지
- 사용법 들을 제공을 해놓았음 
- 터미널 에서 pip install [라이브러리이름]

## 가상환경(venv) : 각 프로젝트마다 서로 다른 파이썬 환경 조성
- 라이브러리와 호환이 되는 파이썬 버전
- 여러 프로젝트가 하나의 pc에서 구동되려면
- 각 프로젝트마다 서로다른 파이썬 버전 필요
- 각 프로젝트마다 개별적으로 라이브러리 설치 필요

- 명령어: python -m venv .venv
- 인터프리터 설정으로 
- "프로젝트/.venv/Script/python" 경로의 파이썬을 지정해준다

- 확인법: which python, which pip
- 프로젝트 경로가 출력되는지 확인
